# 游戏规则

这份文档是当前 Flutter 版电子宠物的规则说明。

## 核心目标

- 还原小时候数码暴龙机那种“要等、要照顾、不能无限刷”的体验
- 保留可爱的原创像素宠物和多分支进化
- 在必要时允许用秘码打破规则，触发隐藏内容

## 当前规则

### 孵化

- 初始是 `数码蛋`
- 默认要等待 `2 分钟`
- 到时间后才能点击 `孵化`
- 孵化后会随机出现幼年期宠物

### 属性

当前主要属性:

- 饥饿
- 心情
- 体力
- 卫生
- 纪律
- 战力

这些属性会随着时间自然变化。

### 行动限制

- `喂食`: 有冷却，太饱时继续喂会有副作用
- `训练`: 有冷却，消耗体力，提高战力和纪律
- `探索`: 有冷却，而且有次数上限，会按时间恢复
- `清洁`: 不是一直都能点，环境够干净时会被禁用
- `休息`: 进入休眠，恢复体力，睡眠中大部分动作会被锁
- `战斗`: 需要成长期以后，且要消耗门票和体力
- `治疗`: 只有生病时可用
- `重启舱`: 默认锁定，需要秘码解锁

## 时间制系统

- 属性衰减按固定时间回合计算
- 关闭页面后再回来，会根据离开的时间补算状态
- 探索次数与战斗门票会随时间恢复
- 睡眠结束也会按真实时间结算

## 失败条件

- 长时间不照顾会累计 `照顾失误`
- 多项关键属性过低会进入 `生病`
- 生病且状态持续恶化，或失误过多，会导致宠物离开

## 进化

进化会受这些因素影响:

- 年龄
- 战力
- 胜场
- 纪律
- 卫生
- 当前分支

当前已经有:

- 普通分支
- 隐藏分支
- 秘码触发分支

## 图鉴

- 每次解锁新形态会记录进图鉴
- 重启后图鉴保留

## 秘码系统

当前内置秘码:

- `HATCH-777`: 立即完成孵化倒计时
- `PROTO-MON`: 下一次孵化固定变成隐藏幼年体
- `OVERCLOCK`: 刷新冷却、补满次数，并提升一点状态
- `RESET-KEY`: 解锁一次 `重启舱`

规则设计意图:

- 平时按正常规则玩
- 想打破规则时，用秘码触发特殊内容

## 当前 UI 形态

现在有两层展示:

- 主界面: 养成、规则、日志、属性
- 悬浮窗预览: 模拟未来后台小窗的视觉样式

说明:

- 现在的悬浮窗还只是界面预览
- 真正的 macOS 后台小窗需要额外接桌面窗口能力
